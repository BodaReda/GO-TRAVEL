# Frontend - Bus Attendance System

React frontend application for the Bus Attendance Management System.

## Installation

```bash
npm install
```

## Running

Development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Features

- React Router for navigation
- Tailwind CSS for styling
- QR code scanning with html5-qrcode
- QR code generation with qrcode.react
- Excel export with XLSX
- Confetti animations
